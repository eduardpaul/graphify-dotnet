## Usage Notes

- **16:9 Hero Images**: Use for blog post headers, documentation landing pages, and GitHub repository social preview (OG image)
- **1:1 Square Images**: Optimal for LinkedIn posts, Twitter/X cards, and social media profile assets
- **AI Platforms**: These prompts are compatible with DALL-E 3, Midjourney, Stable Diffusion, and similar image generation tools
- **Customization**: Adjust colors, logos, and text overlays based on your branding guidelines
- **Licensing**: When generated, ensure compliance with the image generation platform's terms and attribute appropriately if required
